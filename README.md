# Hybrid-Topology-Narval-Scilab


This repository contains the implementation of a Hybrid Network Topology
(Ring + Star + Bus) using Scilab and Narval graph library.

## Features
- Minimum 25 nodes
- Visualization of topology
- Node and edge numbering
- Coloring of nodes and edges
- Analysis of node degree
- Total nodes and edges calculation

## Tools Used
- Scilab
- Narval Graph Library
